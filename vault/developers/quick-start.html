Quick Start

Connect a Jutis wallet, read account state, and sign your first message in under five minutes.

SDK pending release

Jutis SDK is not yet released. This guide uses the browser extension provider API directly — the same interface the SDK will wrap. Integration patterns described here are stable and will not change when the SDK ships.

Installation

No package installation required. The Jutis browser extension injects a provider into every web page it visits.

Users must have the Jutis extension installed. You can prompt them to install it:

if (!window.jutis) { // Show install prompt to user window.open("https://jutis.io", "_blank"); }

Detect Jutis

Always check that the Jutis provider is present before attempting to use it. Use the jutis key on the window object.

// Detect Jutis provider const jutis = window.jutis; if (!jutis) { console.error("Jutis not installed"); // Show install prompt }

Connect Wallet

Call requestAccounts to prompt the user to share their wallet address with your app. This will show a Jutis confirmation popup.

const jutis = window.jutis; async function connectWallet() { try { const accounts = await jutis.requestAccounts(); console.log("Connected:", accounts); // → ["canton1..."] } catch (err) { console.error("Connection rejected", err.code); // err.code === "ACTION_REJECTED" } }

On success, accounts is an array of Canton addresses owned by the user. On rejection, it throws with a code of ACTION_REJECTED.

Read Account State

Once connected, retrieve the signer to read the user's Canton identity and sign operations.

// Get signer instance const signer = jutis.getSigner(); // Get the connected address const address = await signer.getAddress(); console.log("Current address:", address); // → "canton1xyz..." // Get the network chain ID const chainId = await signer.getChainId(); console.log("Chain ID:", chainId); // → "138"

Sign a Message

Request a cryptographic signature from the user. Jutis will display the message content and ask for confirmation before signing.

const signer = jutis.getSigner(); async function signMessage(message) { try { const signature = await signer.signMessage(message); console.log("Signature:", signature); // → "JutisSigned:abc123..." } catch (err) { if (err.code === "ACTION_REJECTED") { console.log("User rejected signing"); } } } signMessage("Authenticate with My dApp");
