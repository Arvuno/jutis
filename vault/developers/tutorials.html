Beginner schedule 5 min

Connect Jutis to a Web App

Build a connect button that detects the Jutis extension, prompts the user for account access, and displays their Canton address. The foundation for every Jutis-powered dApp.

const jutis = window.jutis; async function connect() { if (!jutis) { alert("Install Jutis to continue"); return; } const accounts = await jutis.requestAccounts(); document.getElementById("address").textContent = accounts[0]; } document.getElementById("connectBtn").onclick = connect;

Read Wallet State

After connecting, read the user's address, current chain, and network configuration. Useful for personalizing the UI and validating the right network before transactions.

const signer = jutis.getSigner(); // Get current address const address = await signer.getAddress(); // Get chain configuration const network = await jutis.getNetwork(); console.log("Chain:", network.name); // → "Canton Mainnet" // Listen for network changes jutis.on("chainChanged", (chainId) => { console.log("Switched to chain:", chainId); });

Sign a Message

Request a cryptographic signature from the user for authentication or data integrity. Jutis shows the raw message content before signing — no silent approvals.

const signer = jutis.getSigner(); // Build authentication message const message = [ "Sign this message to authenticate with Acme dApp", "Nonce: 12345", "Expiry: 60 seconds" ].join("\n"); try { const sig = await signer.signMessage(message); verifySignature(sig, message, address); // implement verify } catch (err) { if (err.code === "ACTION_REJECTED") { console.log("User declined to sign"); } }

All levels schedule 8 min

Handle Wallet Errors

Build a resilient integration by handling every error code the Jutis provider can return. Good error handling turns a broken flow into a helpful user experience.

async function robustConnect() { try { const accounts = await jutis.requestAccounts(); return accounts[0]; } catch (err) { switch (err.code) { case "ACTION_REJECTED": showMessage("Connection cancelled. Click connect to try again."); break; case "CHAIN_NOT_FOUND": showMessage("Canton network missing. Add it in Jutis settings."); break; default: showMessage("Something went wrong. Please try again."); } } }
1

Wrap all provider calls in try/catch

2

Handle ACTION_REJECTED gracefully

3

Detect CHAIN_NOT_FOUND and guide user

4

Subscribe to disconnect to clean up sessions
