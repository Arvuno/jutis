Design Reference v1.0

Design Principles

touch_app

Touch-First

Minimum 48px touch targets, generous spacing, thumb-friendly navigation zones.

light_mode

Spacious

Not cramped. More whitespace than desktop. Content breathes on small screens.

smartphone

Native Feel

Mobile-native patterns, not desktop shrunk. Use bottom navigation, sheets, gestures.

shield

Trust & Security

Security visible and prominent. Backup flows are clear. Trust signals on every screen.
